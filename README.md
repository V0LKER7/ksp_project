# Python код

-   `./Код/full_flight.py` — код для сбора информации по всему полёту
-   `./Код/model_comp.py` — код для расчёта физ модели и построение графика сравнения
-   `./Код/plot_for_json.py` — построение графиков скорости и высоты
-   `./Код/speed_hight.py` — сбор данных о высоте и скорости во время взлёта
